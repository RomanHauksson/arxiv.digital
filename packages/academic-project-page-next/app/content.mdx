## Using the Components

Here are examples of all the available components in this template:

### Authors

Use the `Authors` component to display author information with optional links and institutions:

<Authors authors={[
  { name: "Jane Doe", url: "https://example.com", institution: "University of Example", notes: ["*"] },
  { name: "John Smith", institution: "Another University", notes: ["*", "†"] }
]} />

### Links

Add action buttons with the `Links` component:

<Links links={[
  { name: "Paper", url: "https://arxiv.org", icon: "ri:file-pdf-2-line" },
  { name: "Code", url: "https://github.com", icon: "ri:github-line" },
  { name: "arXiv", url: "https://arxiv.org", icon: "academicons:arxiv" }
]} />

### Notes

Display footnotes with the `Notes` component:

<Notes notes={[
  { symbol: "*", text: "Equal contribution" },
  { symbol: "†", text: "Corresponding author" }
]} />

### Highlighted Section

Use `HighlightedSection` to create full-bleed background sections:

<HighlightedSection>

## Abstract

This template provides a comprehensive set of components for building academic project pages. It's built with **Next.js**, **Tailwind CSS**, and **MDX**.

</HighlightedSection>

### Picture

Display responsive images with the `Picture` component:

<Picture src="/dogs-msillm.jpg" alt="Example image" />

### Wide Layout

Use `Wide` to make content wider than the text column:

<Wide>
  <Picture src="/dogs-diffc.png" alt="Wide example image" />
</Wide>

### Figure with Caption

Wrap images in a `Figure` component with a caption:

<Figure 
  figure={<Picture src="/dogs-true.png" alt="Example" />}
  caption={<>A <em>caption</em> for the figure with formatting.</>}
/>

### Two Columns

Use `TwoColumns` for side-by-side layouts:

<TwoColumns 
  left={<Picture src="/dogs-msillm.jpg" alt="Left column" />}
  right={<Picture src="/dogs-diffc.png" alt="Right column" />}
/>

### Video

Embed videos with autoplay, loop, and controls:

<Video src="/outside.mp4" />

### YouTube Video

Embed YouTube videos:

<YouTubeVideo videoId="dQw4w9WgXcQ" />

### Comparison Slider

Compare two images with a draggable slider:

<Comparison 
  itemOne={<Picture src="/dogs-true.png" alt="Before" />}
  itemTwo={<Picture src="/dogs-diffc.png" alt="After" />}
/>

### Tabs

Organize content with tabs (Shadcn UI):

<Tabs defaultValue="tab1">
  <TabsList>
    <TabsTrigger value="tab1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab2">Tab 2</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">
    Content for tab 1.
  </TabsContent>
  <TabsContent value="tab2">
    Content for tab 2.
  </TabsContent>
</Tabs>

### Carousel

Display a swipeable carousel of content:

<Carousel>
  <Picture src="/dogs-msillm.jpg" alt="Slide 1" />
  <Picture src="/dogs-diffc.png" alt="Slide 2" />
  <Picture src="/dogs-true.png" alt="Slide 3" />
</Carousel>

## Getting Started

1. Edit `app/content.mdx` to add your content
2. Place images in the `public/` folder
3. Customize styles in `app/globals.css`
4. Run `npm run dev` to start the development server

Press **"d"** to toggle between light and dark mode!
